# ミナミナ HP — 公開と編集のしくみ

## フォルダの中身
- `index.html` … サイト本体（デザイン）
- `data/site.json` … 編集できる中身（お知らせ・商品・店舗情報など）
- `admin/` … コンテンツ管理画面（ログインして編集する画面）
- `og-image.png` … SNSで共有したときに出る画像
- `robots.txt` / `sitemap.xml` … 検索エンジン向けの設定

## 自分で編集できる場所
管理画面（公開後の `あなたのサイトURL/admin/`）にログインすると、フォーム形式で次を編集できます。
- ヒーローのキャッチコピー・説明文・数字
- お店紹介の本文
- おしらせ（追加・編集・削除）
- おすすめ商品（追加・編集・削除）
- インスタのアカウント・URL
- 店舗情報（住所・電話・営業時間など）
- お問合せの電話番号

保存すると自動でサイトに反映されます（数十秒〜数分）。デザインやレイアウトは崩れません。

## 公開のながれ（初回だけ設定が必要 / 一緒に進めます）
1. GitHub の無料アカウントを作成し、このフォルダを置くリポジトリを用意
2. Netlify を GitHub と連携して公開（以後、内容を変えると自動で再公開）
3. 管理画面のログイン設定
4. 公開後、ファイル内の `example.netlify.app` と `YOUR_GITHUB_USERNAME/minamina-site` を実際の値へ差し替え

## まだ仮の箇所（確定したら差し替え）
- 住所・電話番号・最寄り駅
- 商品名・価格
- セルフエステの料金
- Googleマップの埋め込み
- お問合せフォーム（Googleフォーム連携）
